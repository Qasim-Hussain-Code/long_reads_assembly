# QUAST Quality Assessment

This directory contains QUAST assembly statistics.

## Expected Output Files

- `quast_results/report.html` - Interactive HTML report
- `quast_results/report.txt` - Text summary
- `quast_results/report.tsv` - Tab-separated values

## Key Metrics

- N50, N75, L50, L75
- Total length
- Number of contigs
- GC content
- Gene predictions (Glimmer)
