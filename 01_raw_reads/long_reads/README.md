# Raw Long Reads

This directory contains raw Oxford Nanopore long read sequencing data.

## Expected Files

- `*.fastq.gz` - Raw long read FASTQ files (gzipped)

## Note

Large FASTQ files (>100MB) are not tracked by git. Please download or generate your own sequencing data and place it in this directory.

## File Naming Convention

Example: `sample_name_long_reads.fastq.gz`
